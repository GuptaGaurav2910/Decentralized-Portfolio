Project Overview

The goal of this project is to build a decentralized portfolio that leverages blockchain technology to enhance security, transparency, and control over personal portfolio data. While the core non-blockchain features are complete, the blockchain integration is still under development.

Features Implemented

User Interface: A responsive and intuitive UI for managing portfolio data.

Portfolio Management: Functions to add, edit, and delete portfolio items.

Data Visualization: Charts and graphs to visualize portfolio performance.

In Progress

1) Blockchain Integration: Work is ongoing to integrate blockchain functionalities for decentralized storage and transactions.
2) Smart Contracts: Development of smart contracts to handle portfolio transactions and data security.
3) Decentralized Authentication: Implementing decentralized identity and authentication mechanisms.
